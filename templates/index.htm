Описание сервера

Доступные адреса

  • / - справка (вы сейчас здесь)
  • /map - данные о сотоянии карты в формате JSON
  • /reg/{никнейм} - регистрация в системе, возвращает ваш уникальный идентификатор (id). Повторные запросы меняют ваш никнейм
  • /score - страница общего счёта
  • /about_me/{id} - данные об игроке (id - ваш идентификатор полученный через /reg/{никнейм})
  • /place_gift/{id} - оставить подарок в домике поблизости (id - ваш идентификатор полученный через /reg/{никнейм}.) Подарок будет положен во все домики, квадрат расстояния до которых не превышает {{cfg.GIFT_RANGE_SQUARE}}. Если домиков поблизости нет будет возвращена строка {{cfg.BAD_GIFT_POSITION}}. Если во всех ближайших домиках уже есть подарки будет возвращена строка {{cfg.ALREADY_GIFTED}}. Если удалось одарить хоть кого-то будет возвращена строка {{cfg.SUCCESS}}.
  • /move/{id}/{код_направления} - Перемещает игрока в заданном направлении на {{cfg.PLAYER_STEP}}. Коды направлений: {Верх - {{cfg.UP}}, Вниз - {{cfg.DOWN}}, Влево - {{cfg.LEFT}}, Вправо - {{cfg.RIGHT}} }
  • /view - страница отображения карты. Содержит скрипт отображения draw.js.
  • /base_strategy - страница базовой стратегии. Содержит скрипт базовой стратегии actions.js. (Предполагается, что вы создадите копию этой страницы и будете изменять этот скрипт для достижения лучшего результата).

Важно Интервал между запросами требующими передачу вашего {id}, исключая /about_me/{id} должен быть не менее {{cfg.REQUEST_INTERVAL_MS}} миллисекунд. В случае слишком частых запросов вы будете получать в ответ строку {{cfg.REQUEST_REJECTED}}.

Инфо Базовая стратегия делает так: змейкой спускаемся с текущего положения вниз и на каждом шаге пытаемся положить подарок.

Инфо Если передать неправильный {id} будет возвращён ответ {{cfg.BAD_ID}}.
